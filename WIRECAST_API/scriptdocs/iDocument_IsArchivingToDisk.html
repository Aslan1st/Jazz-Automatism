Method IsArchivingToDisk Interface

Visual Basic: 

IsArchivingToDisk() As Int

C Variants: 

int IsArchivingToDisk();

Method Result

Returns 1 if Wirecast is currently archiving to disk.
Returns 0 if Wirecast is not currently archiving to disk.

Discussion

This function returns whether Wirecast is currently archiving to disk.
