TransitionSpeed Property Type

TransitionSpeed is a string.

Acceptable values

  • "slowest" = the slowest possible transition speed

  • "slow" = the slow transition speed

  • "normal" = the normal transition speed

  • "fast" = the fast transition speed

  • "fastest" = the fastest transition speed

Discussion

This property maps to the menu item "Switch --> Transition Speed".
