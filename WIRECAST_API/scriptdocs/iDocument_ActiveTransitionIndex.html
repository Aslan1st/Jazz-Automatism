ActiveTransitionIndex Property Type

ActiveTransitionIndex is an int.

Acceptable values

  • 1 = The first popup is active

  • 2 = The second popup is active

Discussion

This property defines which of the two transition poups buttons are active.
The transition popup buttons can be found on the Wirecast document window just below the Preview, and to the left of the "Go" button.
