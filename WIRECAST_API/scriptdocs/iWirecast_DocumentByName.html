Method DocumentByName Interface

Visual Basic: 

DocumentByName(name as String, compare_method as Int) As Object

C Variants: 

IDispatch *DocumentByName(const char *string, int compare_method);

name

The name to compare against each document name.

This is the name as seen in the title of the window.

compare_method

The comparison method, which is one of these values:

  • 0 = Exact Match - The string you pass in exactly matches the document name

  • 1 = Contains - The string you pass in is a substring of the document name

  • 2 = Case Insensitive - The lowercase of the string you pass in exactly matches the lowercase of the document name

  • 3 = Case Insensitive Contains - The lowercase of the string you pass in is a substring of the lowercase of the document name

Method Result (Object / IDispatch)

Returns a Wirecast Document Object.
Return 0 if index is out of bounds, or document cannot be found.

Discussion

This function returns a Document Object.
