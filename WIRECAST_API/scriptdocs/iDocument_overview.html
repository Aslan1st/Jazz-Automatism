Getting the Object

You can get a copy of this object by calling the Wirecast Interface DocumentByIndex method.

Methods

  • Broadcast
    Begin broadcasting to the network.

  • IsBroadcasting
    Is Wirecast currently broadcasting to the network.

  • ArchiveToDisk
    Begin archiving to disk.

  • IsArchivingToDisk
    Is Wirecast currently archiving to disk.

  • LayerByIndex
    returns a Document Layer Object.

  • LayerByName
    returns a Document Layer Object.

  • ShotByShotID
    returns a Shot Object.

  • ShotIDByName
    Returns the shot ID for a named shot.

  • SaveSnapshot
    Takes a snapshot of the current output stream for the document and saves image to disk.

  • RemoveMedia
    Removes the media from the document. Does not check if media is still in a Shot.

Properties

  • TransitionSpeed
    The current transition speed (as a string, "slow", "fast", etc).

  • AutoLive
    Whether AutoLive is on (1) or off(0).

  • ActiveTransitionIndex
    Which transition popup is currently active.

  • AudioMutedToSpeaker
    Whether output to the speaker/headphone is muted (1) or not muted (0).
