Method SaveSnapshot Interface

Visual Basic: 

SaveSnapshot(path as String)

C Variants: 

void SaveSnapshot(const char *path);

path

Path (including file name) to file where snapshot should be saved.

Method Result

Does not return a result.

Discussion

This method instructs Wirecast to save a snapshot (single frame) of the current output for this document. The output is a PNG image file.

Each document has a current output even if it is not broadcasting or saving to disk. You may use this method regardless of whether Wirecast is broadcasting or saving to disk.
