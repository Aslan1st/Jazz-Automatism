Describes Wirecast OLE API

This documentation describes the API (Application Programming Interface) for calling Wirecast from your favorite Windows language.

It is written to be as brief as possible to enable you to quickly start interacting with Wirecast via this API.

If you are familiar with how to use your language with an COM/OLE/Automation based API, then you should be able to skip directly to the language agnostic documentation.

Audience / Assumptions

It is assumed that the reader knows how to write in their desired language, and use that languages tools (compiler, interpreter, etc).

However, along with this documentation you will find example code in Visual Basic, C#, Perl to help ease the process of using Wirecast with your favorite language.

Other languages

If an example for your chosen language does not yet exist, you should have a look around the internet to see how to make OLE calls in your language. If you can point us at documentation for this, we will attempt to include your language in a future release of this documentation.
