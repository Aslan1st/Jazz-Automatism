Getting the Object

Your language should support some way of using "GetActiveObject" or "GetObject", which uses the application name to look in the windows registry to find the actual GUID.

If you need the GUID for Wirecast, it is currently (as expressed in C): 

const GUID IDD_Wirecast_app	= { 0xa416c15b, 0xa073, 0x4994, { 0x8d, 0xb5, 0x87, 0x52, 0x7a, 0x41, 0xb2, 0xfa } };
const char IDD_Wirecast_app_str[] = "{A416C15B-A073-4994-8DB5-87527A41B2FA}";
