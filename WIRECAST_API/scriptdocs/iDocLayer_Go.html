Method Go Interface

Visual Basic: 

Go()

C Variants: 

void Go();

Discussion

This brings the current state of the document layer live.

This method is mostly used when the AutoLive property of the Document Object is 0 (off).

If the AutoLive property of the Document Object is 1 (on), then most changes are automatically brought live.

This "go" function works only on the document layer that it is called for, which allows you to bring only one layer at a time live.
