ActiveShotID Property Type

ActiveShotID is an int.

Acceptable values

Any shot_id returned from the document layer functions, such as ShotIDByIndex or ShotIDByName.

Discussion

Setting the ActiveShotID is equivalent to clicking on a shot in the Wirecast interface.

If the AutoLive property of the Document Object is 0 (off), then this function does not change the current output of Wirecast. Use the Go method to make this change live.
