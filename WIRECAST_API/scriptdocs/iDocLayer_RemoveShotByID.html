Method RemoveShotByID Interface

Visual Basic: 

RemoveShotByID(shot_id As Int)

C Variants: 

void RemoveShotByID(int shot_id);

shot_id

An ID retrieved from various functions, such as ShotIDByName and ShotIDByName.

Discussion

Removes the shot from the layer, and it can no longer be referenced. Any subsequent calls through to the shot-api using this shot_id will fail.

If the shot is currently live, Wirecast will continue to broadcast the shot until a new shot is brought live.
