Method DocumentByIndex Interface

Visual Basic: 

DocumentByIndex(index as Int) As Object

C Variants: 

IDispatch *DocumentByIndex(int index);

index

The 1-based index (1..N) of the document to retrieve.

Method Result (Object / IDispatch)

Returns a Wirecast Document Object.
Return 0 if index is out of bounds, or document cannot be found.

Discussion

This function returns a Document Object.
