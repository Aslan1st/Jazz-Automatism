Method ShotByShotID Interface

Visual Basic: 

ShotByShotID(shot_id as Int) As Object

C Variants: 

IDispatch *ShotByShotID(int shot_id);

shot_id

An ID retrieved from various functions, such as ShotIDByName and ShotIDByName.

Each shot in Wirecast has a unique ID which can be used to identify the shot.
This function allows you to get a shot from the document by its ID.
A shot_id is unique across the instance of Wirecast; if you quit Wirecast, the shot_id is not longer valid.

Method Result (Object / IDispatch)

Returns a Wirecast Shot Object.
Return 0 if index is out of bounds, or document cannot be found.

Discussion

This function returns a Shot Object.
