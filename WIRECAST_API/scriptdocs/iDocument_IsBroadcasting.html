Method IsBroadcasting Interface

Visual Basic: 

IsBroadcasting() As Int

C Variants: 

int IsBroadcasting();

Method Result

Returns 1 if Wirecast is currently broadcasting.
Returns 0 if Wirecast is not currently broadcasting.

Discussion

This function returns whether Wirecast is currently broadcasting to the network.
