AudioMutedToSpeaker Property Type

AudioMutedToSpeaker is an int.

Acceptable values

  • 0 = Output to speaker/headphone is not muted (it is audible)

  • 1 = Output to speaker/headphone is muted (it is not audible)

Discussion

This property defines whether the output the headphones is muted or not. It is equivalent to opening "Master Audio" on the "Layout" menu in Wireast and press the headphones button.
