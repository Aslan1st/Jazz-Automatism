Getting the Object

You can get a copy of this object by calling the Document Interface LayerByName or LayerByIndex methods.

Methods

  • ShotCount
    Returns the number of shots that are in the document layer.

  • ShotIDByIndex
    Returns a shot_id for the given 1-based index.

  • ShotIDByName
    Returns a shot_id for a shot whose name matches the given name.

  • PreviewShotID
    Returns a shot_id for the shot on this layer that is currently in preview

  • LiveShotID
    Returns a shot_id for the shot on this layer that is currently live

  • AddShotWithMedia
    Adds a new shot on the layer using the given media filename as source media.

  • RemoveShotByID
    Removes the given shot from the layer.

  • Go
    Causes the layer to be brought Live.
    This only makes brings changes "live" when you have turned AutoLive off (set to 0).
    If AutoLive is on, you generally do not need to use "go".

Properties

  • Visible
    Whether the current layer is visible.
    If AutoLive is off (set to 0), you must use Go after calling this method.

  • ActiveShotID
    Sets the active shot for the layer.
    If AutoLive is off (set to 0), you must use Go after calling this method.
