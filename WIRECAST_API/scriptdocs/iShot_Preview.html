Method Preview Interface

Visual Basic: 

Preview() As Int

C Variants: 

int Preview();

Method Result

  • 0 = Shot is not currently in preview

  • 1 = Shot is currently in preview

Discussion

In Wirecast, this value represents whether the shot is being displayed in the Preview view, the display output to the left of the Live view.

The user can see the Preview by going to the Menu "Layout" --> "Preview"
