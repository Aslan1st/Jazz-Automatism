Method RemoveMedia Interface

Visual Basic: 

RemoveMedia(path as String)

C Variants: 

void RemoveMedia(const char *path);

path

Path (including file name) to file that should be removed.

Method Result

Does not return a result.

Discussion

This method instructs Wirecast to remove the media from the document.

Wirecast does not check if the media exists in any shots. If you use a shot whose media has been deleted, the output will be blank.
