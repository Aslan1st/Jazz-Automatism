Name Property Type

Name is a String.

Acceptable values

Any name you would like to give to the shot.

Discussion

Setting the name is functionally equivalent to selecting "Rename Shot" from the application "Edit" menu in Wirecast.

The name returned is the name as shown in the shot list (the area below the preview in the main document window).
