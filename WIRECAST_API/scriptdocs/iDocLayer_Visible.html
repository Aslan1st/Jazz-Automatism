Visible Property Type

Visible is an int.

Acceptable values

  • 0 = The document layer is not visisble

  • 1 = The document layer is visible

Discussion

This property defines the visibility of a layer.

In Wirecast it is functionally equivalent to the small red Xs which appear when you click on the layer icon in the Layer Panel.

You can open the Layer Panel by selecting "Layer Panel" from the "Layout" application menu in Wirecast. Clicking the checkbox to the left of the layer name "Master Layer 1" (or others) will turn the visilbity of the layer on and off.

If the AutoLive property of the Document Object is 0 (off), then this function does not change the current output of Wirecast. Use the Go method to make this change live.
