Method ShotIDByName Interface

Visual Basic: 

ShotIDByName(name as String, compare_method as Int) As Int

C Variants: 

int ShotIDByName(const char *name, int compare_method);

name

The string to use to compare against each shot name.

compare_method

The comparison method, which is one of these values:

  • 0 = Exact Match - The string you pass in exactly matches the shot name

  • 1 = Contains - The string you pass in is a substring of the shot name

  • 2 = Case Insensitive - The lowercase of the string you pass in exactly matches the lowercase of the shot name

  • 3 = Case Insensitive Contains - The lowercase of the string you pass in is a substring of the lowercase of the shot name

Method Result

Returns a Wirecast shot_id, which can be used in various methods such as ShotByShotID
Return 0 if no shot matches.
If mutiple shots match, a random one is returned.

Discussion

Each shot in Wirecast has a unique ID which can be used to identify the shot.
A shot_id is unique across the instance of Wirecast; if you quit Wirecast, the shot_id is not longer valid.

This searches only on the document layer for a shot which matches the name. To search the whole document, you may use the Document Object ShotIDByName method.

You can make the current shot live by setting the ActiveShotID Property.
You can get other properties about a shot by getting the shot object itself through the Document's ShotByShotID method.
