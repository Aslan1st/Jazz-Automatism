Method AddShotWithMedia Interface

Visual Basic: 

AddShotWithMedia(filepath As String) As Int

C Variants: 

int AddShotWithMedia(const char *filepath);

filepath

The full path to the media file on disk.

Method Result

Returns a Wirecast shot_id, which can be used in various methods such as ShotByShotID
Return 0 if media cannot be loaded.

Discussion

If the file referenced by filename exists and is a media file format that Wirecast can read, then this method adds a new shot to the document layer.

If this function returns 0 for a specific type of media, please verify that the media can be loaded with Wirecast via the GUI.
